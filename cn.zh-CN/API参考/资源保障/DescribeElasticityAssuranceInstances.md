# DescribeElasticityAssuranceInstances

调用DescribeElasticityAssuranceInstances查询弹性保障服务已匹配的运行状态的实例列表。

## 接口说明

当弹性保障服务失效后，实例与弹性保障服务对应的私有池的匹配关联数据也会失效。此时调用该接口查询已失效的服务，返回值为空。
