# 步骤6：测试网络连通性

您可以登录ECS实例测试网络的连通性，以保证配置的IPv6地址已具备公网访问能力。

本示例中，将通过ping -6（或者ping6）命令测试ECS实例与`aliyun.com`的连通性。

1.  远程连接已配置IPv6地址的ECS实例。

    具体操作请参见[通过密码或密钥认证登录Linux实例]()。

2.  运行以下命令验证网络连通性。

    ```
    ping -6 aliyun.com
    ```
