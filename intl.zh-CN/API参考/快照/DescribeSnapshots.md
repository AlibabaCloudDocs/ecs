# DescribeSnapshots

调用DescribeSnapshots查询一台ECS实例或一块云盘所有的快照列表。

## 接口说明

`InstanceId`、`DiskId`和`SnapshotIds`不是必需的请求参数，但是可以构建过滤器逻辑，参数之间为逻辑与（And）关系。

通过阿里云CLI调用API时，不同数据类型的请求参数取值必须遵循一定的格式要求，详情请参见[CLI参数格式说明](~~110340~~)。
