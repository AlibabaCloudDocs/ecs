# DescribeInstanceMonitorData

调用DescribeInstanceMonitorData查询一台ECS实例的监控信息。可查询的指标包括ECS实例的vCPU使用率、突发性能实例积分、接收的数据流量、发送的数据流量、平均带宽等。

## 接口说明

调用该接口时，您需要注意：

-   每次最多返回400条监控数据，如果指定的参数`(EndTime - StartTime)/Period`大于400时，则返回错误。
-   单次最多能查询近30天内的监控信息，如果指定的参数StartTime超过30天，则返回错误。
-   当返回信息中缺少部分内容时，可能是由于系统没有获取到相应的信息，比如当时实例处于已停止（Stopped）状态。
